# Coffee Under Palms — Café Website

Cinematic one-page site for **Coffee Under Palms (CUP)** — a slow-coffee café in Vasant Kunj, New Delhi.

> _Good coffee. No rush. A space to pause._

## Stack

- Hand-written **HTML / CSS / JavaScript** — no build step, no framework.
- Google Fonts (Fraunces, Cormorant Garamond, Inter, DM Mono).
- Unsplash imagery loaded directly from their CDN.

## Files

| File | Purpose |
|---|---|
| `index.html` | Page markup, sections, copy. |
| `styles.css` | Design system, layout, animations. |
| `script.js` | Loader, scroll reveals, magnetic CTAs, reservation form, modals. |
| `logo.webp` | CUP brand mark (used in nav, loader, favicon). |

## Run locally

Any static server works. For example:

```bash
python3 -m http.server 5173
# then open http://localhost:5173
```

## Sections

1. Hero — palm boulevard with golden sun rays.
2. Story / brand pillars.
3. Signature menu items.
4. Full menu (filterable).
5. Spaces gallery.
6. Reservation form.
7. Visit / contact.
